# Point-of-Sale-System
This porject was developed as a course requirement for DCIT 26 - Application Development and Emerging Technologies at Cavite State University - Naic.
## Project Description
A web-based point of sale system built using PHP and MySQL to manage sales transactions, product inventory and receipts for a small pastry shop.
## Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript
- Apache (XAMPP)
## Disclaimer
This system is for academic use only and is not intended for real-world deployment without further development and security enhancements.
